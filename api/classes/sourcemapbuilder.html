applySourceMap

  • applySourceMap(other: SourceMapBuilder, file?: string): void
    • Defined in lib/utility/sourceMap.ts:553

    基于指定的源映射更新当前源映射的源码位置。

    remark

    假如有源文件 A，通过一次生成得到 B，其映射表记作 T。 现在基于 B，通过第二次生成得到 C，其映射表记作 M。 那么就需要通过调用 M.applySourceMap(T), 将 M 更新为 A 到 C 的映射表。

    Parameters

    • other: SourceMapBuilder

      要应用的源映射。

    • Optional file: string

      other 对应的源文件。如果未提供将使用 other.file。

    Returns void

computeLines

  • computeLines(start?: number, end?: number): void
    • Defined in lib/utility/sourceMap.ts:628

    自动补齐指定行的映射点。

    remark

    由于源映射 v3 不支持根据上一行的映射推断下一行的映射。 因此在生成源映射 v3 时，必须插入每一行的映射点。 此函数可以根据首行信息自动推断下一行的信息。

    Parameters

    • Default value start: number = 0

      开始补齐的行号。

    • Default value end: number = this.mappings.length

      结束补齐的行号。

    Returns void
