digo

digo 是一个基于规则流程的自动化构建引擎。

通过插件，digo 可以为项目提供代码预编译、模块依赖打包、压缩优化等构建功能。

特性

  1. 易上手: 没有复杂的概念，您只需简单地定义一些构建的规则流程，立享自动化发布的快感。
  2. 高性能: 得益于 Node.js 的异步 IO 特性，digo 可以高效、并行地构建项目。
  3. 易扩展: 使用插件，轻松满足不同的发布需求。

下载和安装

$ npm install digo -g

如果安装失败，请点击这里。

快速上手

1. 在项目根目录下新建名为 digofile.js 的文件：

var digo = require("digo");

exports.hello = function() {
    digo.src("*.txt")
        .pipe(function(file) { 
            file.content += "哈哈"; 
        })
        .dest("_out");
};

3. 运行 digo:

$ digo hello

执行后项目内所有 *.txt 文件都会拷贝到 _out 目录，且末尾追加了“哈哈”。

如果执行报错请点这里。

digofile.js 示例

var digo = require("digo");

exports.build = function() {
    var list = digo.src("src");
    list.src("*.js").pipe("digo-babel");
    list.src("*.less").pipe("digo-less").pipe("digo-autoprefixer");
    list.dest("_build");
    return list;
};

exports.publish = function() {
    var list = exports.build();
    list.src("*.js").pipe("digo-uglify-js");
    list.src("*.css").pipe("digo-clean-css");
    list.dest("_release");
};

exports.watch = function() {
    digo.watch(exports.build);
};

exports.default = exports.watch;
